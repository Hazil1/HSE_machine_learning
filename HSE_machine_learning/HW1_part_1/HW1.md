# Результаты выполненной работы
1. Был выполнен EDA в результате было выявлено:
   1. В датасете 1005 пропущенных значений, 493 строки полностью задублировано, что составляет 7% от всего train датасета. 
   2. Датасет состоит из 5 текстовых признаков, 4 числовых и 4 категориальных признака.
   3. В признаке `selling_price` есть значение 10_000_000, что может являться выбросом. Также в `km_driven` значение 2_360_457 скорее всего тоже выброс.
   4. Есть дисбаланс в признаках `fuel`, `seller_type`, `transmission`, `owner`.
   5. Высокая отрицательная корреляция между признаками `year` и `km_driven`, `selling_price` и `km_driven`.
   6. Высокая положительная корреляция между признаками `selling_price` и `year`, `selling_price` и `transmission`, `selling_price` и `transmission`.
   7. В выборке более старых автомобилей больше, чем новых.
   8. В выборке есть очень дорогие автомобили, из-за чего среднее выше медины.

2. Была проведена подготовка признаков и датасета.
   1. Были убраны единицы измерения для признаков ``mileage, engine, max_power``.
   2. Предобработан признак `torque`, разделен на два признака `torque` и `max_torque_rpm`.
   3. Были заполнены пропуски в данных медианным значением по каждому признаку.
   4. Было произведено преобразование типов данных.
   5. Была произведена кодировка признаков методом OHE
   6. Данные были отмасштабированные с помощью `StandardScaler`

3. Произведен анализ корреляции между признаками. 

4. Обучены модели линейной регрессии:
   1. Без регуляризации
   2. С L1 и L2 регуляризацией
   3. Только с вещественными признаками и с категориальными признаками.
   4. Проведен feature engineering.
   5. Выполнен подбор гиперпараметров с помощью `GridSearchCV`
   6. Лучшая модель сохранена в `.pickle`-файле

Наилучшей моделью оказалась модель с категориальными признаками с L2 регуляризацией.
